2.Ezwww

4.Welcome

题目给了hints。

hints1：In addition to the GET request method, there is another common request method… hint2: Try to use burpsuite to do this challenge

所以用burpsuite改包，将GET改为POST，网页回显源码

典型的数组漏洞，将roam1和roam2以数组形式post上传

roam1[]=1&roam2[]=2

回显出phpinfo()

结果发现在core中有/var/www/html/f1444aagggg.php

然后直接访问http://49.234.224.119:8000/f1444aagggg.php发现是not found，但是样式显然不对劲。查看network在响应头中发现Flag:SYC{w31c0m3_t0_5yc_r0@m_php1}
